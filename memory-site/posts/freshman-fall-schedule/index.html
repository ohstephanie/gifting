TLDR

22-23 academic year -- Freshman fall. my first semester at princeton.

This was a crazy time. A lot of adjusting had to be done and I came into school really scared. My brother had told me that classes were really difficult, and I was scared I wouldn't be able to keep up. So I signed up for the easiest schedule ever. It's actually laughable how easy it is. Wow...

I remember I cried every day during the very first week of class because I was so nervous. It all started on the first day of class when I accidentally went to McCosh health center instead of McCosh lecture hall and thought my life was over. I dropped that class that afternoon.

I also couldn't get into SOC101, which I desperately wanted to take since I was still considering majoring in sociology. I never ended up getting into the class since it was so popular that year, so I ended up taking EAS 216 instead. I didn't know how to code either, so COS126 also gave me a hard time.

But I had so much fun in my zee group and with my roommates. It was truly such a fun hallway to come back to in the evening, and I loved my RCA too.

COS 126 Computer Science: An Interdisciplinary Approach
EAS 216 Writing and Culture of Premodern Korea
FRS 124 dot dot dash - Exercises in Mark Making
KOR 103 Intensive Korean I
